# 태그팀 서버

태그팀 Express 실습 서버입니다.

## 시작하기

```bash
npm install
npm start
```

서버는 `http://localhost:3000` 에서 실행됩니다.

## API 목록

| 메서드 | 경로       | 설명          |
|--------|------------|---------------|
| GET    | /          | 서버 확인     |
| GET    | /members   | 멤버 목록     |
| GET    | /leader    | 팀장 정보     |
| GET    | /study     | 스터디 정보   |
| GET    | /game      | 게임 목록     |
| GET    | /team      | 팀 정보       |
| GET    | /place     | 활동 장소     |
